 Use the second conditional to talk about imaginary or hypothetical situations in the present and future. A hypothetical situation contrasts with reality.

 if -part                          main part
 If + S+ V2/ed                 S + would + V      
 If + S + didn't + V        S + wouldn't + V 
I would always sort the rubbish if I had a lot of time.
