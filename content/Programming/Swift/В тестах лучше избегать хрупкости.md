Чтобы сделать более стабильными тесты не нужно писать их с хардкодными строками, лучше написать поиск вхождения и так и назвать тесты. 
Так тесты получаются более стабильными. 
```swift
// Fragile
func testUserProfileFormatting() {
let user = User(name: "John Doe", age: 30)
XCTAssertEqual(user.formattedDescription, "Name: John Doe, Ahe: 30")

// Improved test
func testUserProfileContainsNameAndAge() {
let user = User(name: "John Doe", age: 30)
let description = user.formattedDescription
XCTAssertTrue(description.contains("John Doe")
XCTAssertTrue(description.contains("30")
}
```
